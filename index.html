Q2. Write code to explain prototype chaining

                arr = [1,2,3,4];
                console.log(arr._proto.proto.proto_)

The array named arr has prototype chaining as follows: arr -> Oririnal Array -> Original Object -> Null. this figure represents that at first arr._proto_ gives its prototype then at arr._proto.proto_ gives the protype as object of default array again at arr._proto.proto.proto_ it comes to the default Objects prototype which is null.

          console.log(Male.prototype._proto.proto.proto_)

The output here is null. The Male class is extension of Human class. In console statement we first have Male.prototype that gives the output prototype of male.for further Male.prototype._proto_ , this gives the output as protype object for Human class. The continuation of that will be the Object(the default). And finally Object's protype is null.
